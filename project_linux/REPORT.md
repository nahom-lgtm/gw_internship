### Working with Files and Directories

```
$ pwd
--> prints out the path to working or current directory

$ mkdir ~/project-1
creates a directory named "project-1" under the home (~/) direcgtory

$ mkdir ~/project-1/project-linux

$ cd ~/project-1/project-linux

$ pwd

$ touch file.txt

$ ls -ltr

$ cp file.txt file_copy.txt

$ ls -ltr

$ mv file_copy.txt /tmp/

$ rm /tmp/file_copy.txt

$ cd ~

$ rm -r ~/project-1
```
